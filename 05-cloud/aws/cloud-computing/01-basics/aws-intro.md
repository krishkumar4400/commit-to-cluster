## What is AWS?

AWS (Amazon Web Services) is a cloud platform that provides services like:

- Servers
- Storage
- Databases
- Networking

#### in simple words:

“AWS is a platform where you rent computers and services online”

### Used by:

- Startups
Big companies (Netflix, Amazon, etc.)

--- 

### AWS Services

AWS provides many services, but main ones are:

- EC2 (Elastic Compute Cloud):  Virtual server in cloud

You can run apps, websites
Works like your own computer

---

### Example:
Hosting a website

- S3 (Simple Storage Service)
 Storage service

Store files, images, videos
Very secure and scalable

 Example:
Uploading project files or images

---

- RDS (Relational Database Service)

Managed database

Supports MySQL, PostgreSQL
No need to manage manually

Example:
Storing user data

---

- VPC (Virtual Private Cloud)

Private network inside AWS

Controls traffic
Improves security

Simple:

“Your own private internet inside AWS”

---

- Route 53

DNS service

Connects domain name to server

Example:
Typing google.com → connects to server IP
